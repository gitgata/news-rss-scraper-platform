# News RSS Scraper Platform

Advanced web scraper and RSS news aggregator built with FastAPI, BeautifulSoup, feedparser, and HTMX.

## Features
- Scrape news from any website
- Generate custom RSS feeds
- Manage sources
- Modern responsive UI
- Background scraping with APScheduler
- SQLite database
- Docker support

## Quick Start
```bash
pip install -r requirements.txt
uvicorn main:app --reload
```
